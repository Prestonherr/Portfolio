# Preston Herr — Portfolio

A personal portfolio site showcasing my projects, skills, and contact info.

**[prestonherr.dev](https://prestonherr.dev)**

## Tech Stack

- **React 19** — UI framework
- **Vite** — build tool & dev server
- **EmailJS** — contact form email delivery
- **CSS** — custom styling with dark/light theme support
- **ESLint** — code linting
- **Vercel** — deployment & hosting
